Project Overview

Intelligent monitoring for sustainable buildings

The HVAC Carbon Emission Monitoring System is designed to monitor and control the air conditioning systems in commercial buildings. It measures energy usage, indoor air quality (CO₂ levels), and calculates the corresponding carbon emissions in real time.

By analyzing data, the system helps optimize HVAC operation, improve comfort, and support sustainability goals.

How It Works

Five-step intelligent monitoring process

1

Data Collection

Sensors continuously collect energy, temperature, and CO₂ data from the HVAC system.

2

Transmission

ESP32 transmits sensor readings to Raspberry Pi via Wi-Fi in real-time.

3

Processing

Raspberry Pi stores data and calculates energy-based carbon emissions.

4

Visualization

Dashboard displays real-time performance metrics and carbon footprint analysis.

5

Optimization

Control logic adjusts ventilation and cooling for maximum efficiency.
